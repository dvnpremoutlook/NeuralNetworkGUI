# NeuralNetworkGUI
GUI written in C++ for visualization of Neural Networks

Step1: Installation

-Requirements 
  Python 3.10
  MSYS2 Installed 

In MSYS2 you need to install Python.h
  Instructions for that are on the MSYS2 package installer page

Step2: Running 
- Clone the github 
- Run bat file 
- Run a.exe
